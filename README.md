## 🧩 Features

- 🔐 **64-bit Secure Sessions:** Fully compatible with MongoDB & Mega storage.
- ☁️ **Cloud Sync:** Sessions are stored securely for long-term persistence.
- 📱 **Dual Pairing:** Supports both **QR Code** and **Pairing Code** methods.
- 🚀 **Multi-Platform:** One-click deployment for various cloud providers.
